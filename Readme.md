 Tic-Tac-Toe Web Application

A modern and fully responsive Tic-Tac-Toe Web Application built using **HTML, CSS, and JavaScript**.
This project features an interactive gaming interface with futuristic neon UI, smooth animations, game logic, and responsive design.

---

# 🚀 Project Overview

This project was developed as part of a Web Development Internship task to practice:

* JavaScript game logic
* DOM manipulation
* Event handling
* Responsive web design
* Interactive UI/UX development

The application provides a visually engaging gaming experience with both multiplayer and computer gameplay support.

# ⚙️ Functionalities

## ✅ Player Turn Switching

Automatically switches turns between X and O players.

## ✅ Winning Condition Checking

Checks:

* Rows
* Columns
* Diagonals

to determine the winner.

## ✅ Draw Detection

Detects draw situations when all cells are filled without a winner.

## ✅ Restart Functionality

Resets the board and starts a new game instantly.

## ✅ Computer AI Mode

Allows users to play against computer-generated moves.
